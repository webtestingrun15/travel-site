# Demo travel website

An example travel website that was built using modern tools using:

* npm
* webpack
* Gulp
* Browser Sync
* jQuery
* Es6
* Babel
* BEM
* PostCSS
* Git

This site was created from a PSD and converted to HTML and made responsive for desktops, tablets,and mobile devices.
